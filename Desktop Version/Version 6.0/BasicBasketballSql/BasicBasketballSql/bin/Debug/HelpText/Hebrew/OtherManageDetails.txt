אם תבחר/י את האפשרות הזו תוכל/י לקבל את המידע הבא:

המשחקים של הקבוצה שלי -> ייתן לך את כל המשחקים של קבוצתך בליגה.

משחקי הבית שלי -> ייתן לך את כל המשחקים שקבוצתך היא קבוצת הבית.

משחקי החוץ שלי -> ייתן לך את כל המשחקים שקבוצתך היא קבוצת החוץ.

כל שחקני הליגה -> יראה לך את כל השחקנים שיש בליגה.

כל קבוצות הליגה -> יראה לך את כל הקבוצות בליגה.

כל משחקי הליגה -> יראה לך את כל המשחקים שיש בליגה.
